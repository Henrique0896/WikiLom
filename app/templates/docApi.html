Documentação API

A API do WikiLom é um serviço para consultar, escrever, editar e apagar os dados salvos neste aplicativo. Ela funciona independentemente desta plataforma, pois utilizou-se protocolos e padrões para comunicação e transferência de dados. Para utilizar é necessário conhecer apenas a sua interface descrita abaixo:

Listagem Completa

Para obter uma listagem completa (no formato JSON) das matérias cadastradas na aplicação basta inserir:

/api/lista

Lista dos Títulos

A rota abaixo lista apenas os títulos das matérias cadastradas na aplicação

/api/titulos

Filtro

A rota para filtragem recebe dois parâmetros: "campo" e "filtro". O filtro pode ser qualquer texto e o campo deve ser algum da seguinte lista:

  • geral
  • ciclo_de_vida
  • meta_metadados
  • metadados_tecnicos
  • aspectos_educacionais
  • direitos
  • relacoes
  • classificacao
  • conteudo

/api/consulta/"campo"/"filtro"

Adicionar

Para adicionar uma nova matéria utilizando a api, basta entrar com os parâmetros "página" e "paginação". Página é o nome da matéria, exemplo "galo". Como existem muitas páginas referentes a um nome, então devemos adicionar um número que será a paginação. Este número refere a qual página iremos adicionar. Utilizando a plataforma é possível saber exatamente o título da página que irá ser adicionada, mas na api devemos entrar diretamente com o número de paginação.

/api/add/"pagina"/"paginacao"

Atualizar

/api/delete/"titulo"
